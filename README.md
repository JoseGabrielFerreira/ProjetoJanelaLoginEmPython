Projeto Janela de Login em Python

Aplicativo de **cadastro** desenvolvido em **Python** com interface gráfica **Tkinter**.  
Permite registrar **nome, e-mail e senha**, salvando os dados em um arquivo `.txt` com **ID automático**.

 Funcionalidades
- Interface gráfica com Tkinter  
- Campos para nome, e-mail e senha  
- Geração automática de ID  
- Salvamento em arquivo texto (`texto.txt`)

Tecnologias
- Python 3
- Tkinter
- Biblioteca OS

Exemplo
O programa cria um arquivo `texto.txt` com cadastros no formato:
